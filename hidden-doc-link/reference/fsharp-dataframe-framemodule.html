Signature:frame1:Frame<'R,'C> -> frame2:Frame<'R,'C> -> Frame<'R,'C>
Type parameters: 'R, 'C

Append two data frames. The columns of the resulting data frame will be the union of columns of the two data frames. The row keys may overlap, but the values must not - if there is a value for a certain column, at the same row index in both data frames, an exception is thrown.
